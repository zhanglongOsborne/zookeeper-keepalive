typedef zk_hashtable*(* result_checker_fn)(zhandle_t *, int rc)

The function must return a non-zero value if the watcher object can be activated as a result of the server response. Normally, a watch can only be activated if the server returns a success code (ZOK). However in the case when zoo_exists() returns a ZNONODE code the watcher should be activated nevertheless.
